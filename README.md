# [Twippy](https://twippy.netlify.app/)
## ハックツハッカソンスピノカップ
## Description
過去ツイートをタイピングして夏休みを振り返ろう
> 『ハックツハッカソン/スピノ杯』に提出する作品として開発．テーマは『夏休み最終日にだけ使うアプリ』
## Built With
- React
- TypeScript
- Styled-components
- twitter api(backend)
- Express (backend)
